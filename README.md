<h2>주요 기능</h2>

**1. 수입/지출 내역**
<br/>
수입과 지출을 간편하게 생성, 수정, 삭제의 기능을 가지고 있으며, 추가적으로 저장한 수입/지출 내역은 '엑셀'로 내보낼 수 있다<br/>
**2. 수입/지출 보고서**
<br/>
사용자가 작성한 수입/지출 내역은 그래프를 통해 월별, 주별로 자세히 확인 할 수 있다<br/>
**3. 예산**
<br/>
사용자가 카테고리별로 한달 예산을 설정할 수 있다<br/>
